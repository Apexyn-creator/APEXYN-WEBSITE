# Apexyn Solutions Website

A Vite + React + Tailwind CSS site.

## Local development
```
npm install
npm run dev
```

## Build
```
npm run build
```
Outputs static files to `dist/`.

## Deploy on Vercel
1. Push this folder's contents to your GitHub repo (root of repo, not a subfolder).
2. In Vercel, import the repo — it will auto-detect Vite and set the correct build command (`npm run build`) and output directory (`dist`).
3. Click Deploy.
